This is repository for HIMIA dataset.

base can be downloaded from openslr:
AISHELL1: http://www.openslr.org/33/
HI-MIA: http://www.openslr.org/85/

This folder contains two subfolders:
w1: a keyword spotting recipe, using aishell1 asr data for negative data 
    and aishell hi_mia data as positive data
v1: a speaker verification recipe. More can be found in v1/README.md
